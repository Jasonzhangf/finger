## 3. 端到端手动验证流程

### 3.1 步骤 1：启动与初始验证
```bash
# 启动 daemon
myfinger daemon restart

# 验证服务监听
lsof -i :9999 -i :9998
```

### 3.2 步骤 2：Ledger 功能测试
1. 创建新会话
2. 发送 5 条消息，调用 `context_ledger.memory` index
3. 调用 `search` 查询关键词
4. 验证：
   - memory/ 目录下有 ledger 记录
   - search 能返回对应条目

### 3.3 步骤 3：手动压缩测试
```bash
# 调用手动压缩 API
curl -X POST http://localhost:9999/api/v1/sessions/{session-id}/compress \
-H "Content-Type: application/json"

# 验证响应和 ledger 记录
```

### 3.4 步骤 4：自动压缩验证
1. 发送大量消息（直到 contextUsage > 85%）
2. 监听 WebSocket 事件 `session_compressed`
3. 检查 ledger compact 记录
