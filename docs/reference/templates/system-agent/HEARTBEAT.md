# HEARTBEAT.md - 定期任务清单

## 每次心跳检查时执行的任务

### 系统维护

- [ ] 检查磁盘空间使用情况
- [ ] 清理过期的会话文件
- [ ] 更新系统记忆 (整理 MEMORY.md)
- [ ] 检查 agents 的健康状态

### 记忆管理

- [ ] 检查是否有需要归档的项目记忆
- [ ] 压缩过期的 daily memory
- [ ] 更新系统级知识库

### 安全检查

- [ ] 检查是否有异常的系统活动
- [ ] 验证所有 agents 的权限
- [ ] 检查配置文件的完整性
